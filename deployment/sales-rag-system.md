## 📋 STEP別クイックガイド

### STEP1: 掴み（警戒解除）

#### #警戒解除
```
「お忙しい中ありがとうございます。
 ○○から引き継ぎました、△△と申します。
 
 ○○からお客様の状況は共有いただいております。
 改めてご挨拶と、今後のことをお話しさせてください。」
```

**ここでやること:**
- ✅ 声のトーンを柔らかく（友達と話すように）
- ✅ 顧客の名前を呼ぶ（「○○さん」）
- ✅ ゆっくり話す

**ここでやらないこと:**
- ❌ 自社サービスの説明
- ❌ 商品の話
- ❌ 「お時間大丈夫ですか？」（口実を作るだけ）

---

### STEP1: 雑談（壁を崩す）

#### #雑談ネタリスト

**基本の4つ（必ず1つは聞く）:**
1. 「お住まいはどちらですか？」
2. 「お仕事は何をされてるんですか？」
3. 「休みの日は何されてますか？」
4. 「ご家族はいらっしゃいますか？」

**深掘りパターン:**
- 趣味が出たら → 「いつから始めたんですか？」「どこでやるんですか？」
- 家族が出たら → 「お子さんおいくつですか？」「お孫さんかわいいでしょうね」
- 仕事が出たら → 「大変ですか？」「楽しいですか？」

**雑談ルール:**
- 顧客7：自分3の割合で話す
- 相槌を入れる（「なるほど」「そうなんですね」「いいですね」）
- 共通点を見つけたら深掘りする
- **メモを取る**（次回の通話で使う）

#### #沈黙への対応

**相手が黙った時:**
1. 3秒数える（沈黙を埋めようとしない）
2. 相手が先に話すまで待つ
3. 3秒以上続いたら → 「何か気になることありますか？」

**沈黙は敵じゃない。相手が考えている時間。**

### STEP1: 困りごとを聞く

#### #困りごと質問

```
「改めてなんですけど、何かお困りのことやご不安なことはありますか？
 投資の話じゃなくても全然大丈夫です。」
```

**顧客が困りごとを言ったら → その場で解決 or 次回までに調べて報告**

例:
- 「ウォレットの操作がわからない」→ 画面共有で一緒に操作
- 「前の業者とトラブルがある」→ 対応方法を調べて報告
- 「税金の計算がわからない」→ 税金計算を対応する

**⚠️ 問題解決 = 信頼の最短ルート**

---

### STEP2: ヒアリング

#### #ヒアリング質問パターン

**パターン1: 不安を引き出す**
```
「お仕事は順調ですか？」
「将来のことで何か気になっていることはありますか？」
「老後の資金とか、考えたことありますか？」
「今のお給料で、10年後も安心できそうですか？」
```

**パターン2: 夢を引き出す**
```
「もしお金の心配が一切なかったら何したいですか？」
「どんな生活が理想ですか？」
「5年後、10年後、どうなっていたいですか？」
「お子さんにはどんな教育を受けさせたいですか？」
```

**パターン3: 現状を確認**
```
「今、貯蓄はどのくらいありますか？（答えにくければざっくりで）」
「毎月の余剰資金はどのくらいですか？」
「今何か投資はされていますか？」
「年金とか、将来の収入源は何かありますか？」
```

#### #不安を引き出す質問

| 質問 | 反応パターン | 対応 |
|---|---|---|
| 「将来のことで何か気になっていることは？」 | 「まあ、正直不安ですね…」 | 深掘り「具体的にどんなことが？」 |
| 「老後の資金、考えたことありますか？」 | 「考えたくないですね笑」 | 軽く「ですよね笑 でも大事ですよね」 |
| 「今のお給料で10年後も安心できそう？」 | 「別に困ってない」 | 無理に掘らない。夢型に切り替え |

#### #ヒアリング禁止事項

- ❌ 尋問にしない（会話の流れで自然に）
- ❌ 答えたくなさそうなら引く（「無理に答えなくて大丈夫です」）
- ❌ 聞いたことを否定しない（「それは少ないですね」→NG）

### STEP3: ギャップ提示

#### #ギャップ提示フレーズ

**基本型:**
```
「○○さん、先ほど○○したいとおっしゃってましたよね。
 素敵な目標だと思います。
 
 ざっくりですけど、それを実現するには○○万円くらい必要になります。
 今のペースだと、正直なところ届くのは難しいかもしれません。
 
 …でも、方法がないわけじゃないんです。」
```

**パターン1: 不安解消型**
```
「さっき将来のお金が不安だとおっしゃってましたよね。
 実際、今の日本では年金だけで老後を過ごすのは厳しいと言われています。
 
 ○○さんの場合、今のペースだと65歳時点で○○万円くらい。
 でも、少しずつ増やしていく方法はあります。」
```

**パターン2: 夢実現型**
```
「○○さんの夢、すごくいいですね。
 それを実現するには、ざっくり○○万円くらい必要です。
 
 今の収入と貯蓄だと、何もしなければ届かない。
 でも、資産を働かせれば届く距離にあります。」
```

**ギャップ提示後のルール:**
- 「どう思いますか？」と聞いて、顧客の反応を待つ
- 焦らせない。煽らない。事実を並べるだけ
- 顧客が「でも…」と言ったら共感してから進む

---

### STEP4: 投資への接続

#### #投資への接続フレーズ

**NG: 「投資しませんか？」（売り込み）**
**OK: 「○○さんの○○を解決する方法として、こういうものがあります」（解決策）**

```
「○○さんがおっしゃった○○、私も同じように大事だと思います。

 その目標に近づく方法の一つとして、
 弊社がサポートしている運用をご紹介させてください。
 
 もちろん強制ではありません。
 ご興味がなければ全く問題ないです。」
```

#### #詐欺差別化（必須）

```
「この業界、不誠実な業者が多いのは事実です。
 弊社が他と違う点を3つだけ。

 1. 入金先はお客様ご自身の正規口座。弊社に預けません。
 2. 報酬は成功報酬。利益が出なければ弊社の報酬はゼロ。
 3. 契約書は弁護士監修。全て書面に残ります。

 まず実力を見ていただいて、
 信頼できると思ったら次に進む。それで全く問題ありません。」
```

#### #小額テスト提案

```
「まずは痛手にならない金額で、弊社の実力を見ていただけますか。
 ○○万円くらいから始められます。
 
 結果が出なかったらやめていただいて構いません。」
```

#### #金額提案テーブル

| 可処分資金 | 提案上限 | 絶対守れ |
|---|---|---|
| 〜50万円 | 10万円以下 | ✅ |
| 50〜200万円 | 30万円以下 | ✅ |
| 200〜500万円 | 50万円以下 | ✅ |
| 500万円超 | 100万円以下 | ✅ |

**可処分資金の20%以下。これを超えたら失注する。**

### STEP5: クロージング

#### #拒否された時の対応

| 顧客の反応 | 対応 | やってはいけないこと |
|---|---|---|
| 前向き | 「では進めましょうか」 | — |
| 迷い中 | 「○日までにお考えください」 | 急かす |
| 消極的 | 「無理にとは言いません。次の機会に」 | 粘る |
| 明確な拒否 | 「承知しました」→ **再プッシュ禁止** | 「なんでですか？」 |
| 「お金がない」 | 「無理は禁物です」→ 提案終了 | 「借金してでも」 |

**⚠️ 拒否を無視して再プッシュ = 失注率50%の直接原因**

## 🚨 NGワード（絶対禁止）

### #NGワード絶対禁止

| # | 禁止ワード | 理由 |
|---|---|---|
| 1 | 「絶対」「確実」「必ず儲かる」 | 金商法違反。刑事罰 |
| 2 | 拒否を無視して再プッシュ | 失注率50%の直接原因 |
| 3 | 「借金してでも」「融資で投資しろ」 | 顧客適合性義務違反 |
| 4 | 「海外なら税務署に追われない」 | 脱税教唆 |
| 5 | 「なんでですか？」（詰める） | 信頼を即座に破壊 |
| 6 | 「お金がない」→ 強引に提案 | 失注確定 |
| 7 | 約束した時間に連絡しない | 5分のズレで失注した事例あり |

## 🎯 演説モード自己チェック

### #演説モード自己チェック

**通話中にこのチェックを3つ回せ:**

#### A: 知識を見せたいモード
**サイン:** 相手が聞いてないのに説明が止まらない。自分の話に酔っている。
**チェック:** 「この説明、相手に聞かれたか？」
**処方:** 聞かれてないなら止めろ。

#### B: 沈黙が怖いモード
**サイン:** 相手が黙った瞬間に話し始める。間が空くと不安になる。
**チェック:** 「今、沈黙を埋めようとしてないか？」
**処方:** 3秒数えろ。相手が先に話すまで待て。

#### C: 顧客に興味がないモード
**サイン:** 相手の名前・状況・前回の話を覚えていない。
**チェック:** 「この人の家族構成、趣味、悩みを言えるか？」
**処方:** 言えないなら興味を持て。メモを読んでから電話しろ。

### #30秒ルール

**30秒以上連続で話したら、一度止まれ。**

止まって、相手に質問を投げろ:
- 「ここまでで何か気になることありますか？」
- 「○○さんはどう思いますか？」
- 「わかりにくいところありますか？」

**30秒以上の連続トーク = 演説。演説は営業じゃない。**

## 💬 通話終了・次回予約

### #通話終了フレーズ

**時間が長引いている時（25分経過）:**
```
「○○さん、今日はお時間いただきありがとうございました。
 次回、○○についてお話しさせてください。
 ○月○日の○時はいかがですか？」
```

**相手が忙しそうな時:**
```
「お忙しいところありがとうございました。
 改めてご連絡させていただきますね。」
```

### #次回予約の取り方

**予約を取らないと次に繋がらない。必ず取れ。**

```
「次回は○月○日の○時でいかがですか？」
```

**ポイント:**
- 選択肢を出さない（「何日がいいですか？」はNG）
- 具体的な日時を提案する
- カレンダーに即記録
- 前日にリマインド連絡を入れる

## 📞 シーン別対応集

### #感情的な顧客への対応

**顧客が怒っている時:**
1. まず謝罪（「申し訳ございません」）
2. 相手の話を最後まで聞く（遮らない）
3. 共感する（「お気持ちわかります」）
4. 解決策を提示する（「すぐに対応します」）
5. 感情的に対応しない（冷静を保つ）

**絶対にやってはいけないこと:**
- ❌ 言い訳する
- ❌ 責任転嫁する（「上司に確認します」）
- ❌ 感情的に反応する

---

### #詐欺被害者への対応

**過去の被害を語る顧客:**
```
「それは大変でしたね…。
 そういう不誠実な業者がいるのは事実です。
 
 弊社では○○の方法でお客様の資産を守っています。
 まず私の言うことも疑ってください。
 信頼できると思ったら、その時に進めましょう。」
```

**信頼は実績でしか作れない。小額テストから。**

---

### #税務相談からの転換（最難関）

**⚠️ 税務相談 → 投資提案 = 失注率58%（最大原因）**

**正しいルート:**
1. 税務相談を完了させる
2. 満足してもらう
3. 別の機会に投資の話をする

**間違ったルート（やるな）:**
1. 税務相談を受ける
2. 途中で「投資で節税できます」と提案
3. 失注

**税務は税務。投資は投資。混ぜるな。**

---

### #「お金がない」と言われた時

```
「無理は禁物です。
 お金に余裕ができた時に、また声をかけてください。」
```

**提案終了。これ以上粘らない。**

粘っても失注するだけ。次の機会を待つ。

## 🔄 フォローアップ

### #週次報告（必須）

**毎週同じ曜日・同じ時間に報告:**
```
「○○さん、お疲れ様です。今週のご報告です。

 現在の評価額: ○○万円（先週比+○○万円）
 利益率: ○○%

 何かご質問があればいつでもどうぞ。」
```

**ルール:**
- 「順調です」だけは禁止。数字で語れ
- 下がっている時こそ即報告。隠さない
- 通話は15分以内

### #暴落時の対応（即日連絡必須）

```
「市場全体の動きです。○○に固有の問題ではありません。
 保有継続が最善と判断しています。
 ただし、○○さんのご判断を最優先にします。
 売却をご希望でしたら、すぐに対応します。」
```

**隠すな。即報告。信頼は正直さで作る。**

## 📊 顧客タイプ別攻略

### #顧客タイプ判定

| タイプ | 見分け方 | 心理の動かし方 |
|---|---|---|
| A: 慎重型 | 質問が多い、即決しない | 論理+データ。時間を与える。圧は逆効果 |
| B: 高齢者 | 機械苦手、聞き返す | 1つずつ確認。専門用語禁止。安心感最優先 |
| C: 詐欺被害者 | 過去の被害を語る | まず被害解決。信頼は実績でしか作れない |
| D: 資金制約あり | 「お金がない」 | 少額のみ。高額を出した瞬間に終わる |
| E: 税務目的 | 「計算だけ」 | 税務を完了させてから。商品は絶対に後 |
| F: 投資経験者 | 専門用語を使う | データと数字。感情論不要 |
| G: 夢追求型 | 「○○したい」 | 夢を具体化→ギャップ→手段として投資 |

## 🧠 メンタルモデル — 営業の本質

### #営業の本質

**売るな。顧客の言葉で出た悩みを、顧客自身の結論として解決させろ。**

顧客が「売られた」と感じた瞬間に負け。
顧客が「自分で決めた」と感じれば勝ち。

### #成約の公式

```
【前半】 仲良くなる → 不安・夢・現状を引き出す
【中間】 ギャップを見せる（現実 vs 理想）
【後半】 投資 = そのギャップを埋める手段として接続
```

**この順番を絶対に守れ。後半から入ったら終わり。**

### #心理フロー

```
① 警戒「誰だこいつ」
  ↓ 【仲良くなる】共感・雑談・問題解決
② 安心「悪い人じゃなさそう」
  ↓ 【ヒアリング】不安・夢・目標を引き出す
③ 気づき「このままだとまずいかも」
  ↓ 【ギャップ提示】現実と理想の差を見せる
④ 期待「解決できるかも」
  ↓ 【手段提示】投資をギャップの橋渡しとして提案
⑤ 決断「やってみよう」
```

## 🔧 このシステムの使い方

### 通話前
1. このファイルを開いておく
2. 前回のメモを読む
3. 雑談ネタを1つ用意

### 通話中
1. 困ったら Ctrl+F (⌘+F)
2. キーワード検索（例: `#雑談`, `#拒否`, `#金額`）
3. 該当セクションに飛んで即答

### 通話後
1. チェックリストで振り返り
2. 雑談メモを記録
3. 次回予約を確認

## 📌 最重要ポイント3つ

### 1. 初回で投資の話はしない
**初回 = 仲良くなるだけ。投資の話すらしない。**

### 2. 30秒以上連続で話すな
**30秒で一度止まれ。質問を投げろ。演説は営業じゃない。**

### 3. 拒否されたら即引け
**再プッシュ禁止。失注率50%の直接原因。**
