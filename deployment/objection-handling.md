# 反論切り返し集 — 失注53件から抽出

**対象**: 部下4人
**用途**: 顧客の反論に対する切り返しトークを暗記し、通話中に即座に使う
**ベース**: `lost_customers_summary.md`失注53件 + `kamiyama-analysis-report.md`成約18件
**更新日**: 2026-03-29

---

## 使い方

### 準備（通話前）
1. このファイルを印刷して手元に置く
2. 頻出反論TOP10を音読して暗記
3. 顧客タイプを確認し、該当セクションを読む

### 通話中
- 顧客が反論したら、該当パターンを見る
- 切り返しトークをそのまま使う（自己流にアレンジしない）
- **切り返し後に必ず顧客の反応を確認**（「どう思いますか？」）

### 通話後
- 切り返しが効いたか自己採点
- 効かなかった場合はメモに記録（次回の改善材料）

## 顧客タイプ別の反論パターン

### タイプA: 慎重型

**特徴**: 質問が多い、即決しない
**頻出反論**:
- 「今すぐは決められない」
- 「他の業者と比較したい」
- 「結果が出なかったらどうするんですか」

**対策**:
- 論理+データで説明
- 時間を与える（焦らせない）
- 選択肢を出す（「AとBがあります。推奨はA」）

---

### タイプB: 高齢者

**特徴**: 機械苦手、警戒心強い
**頻出反論**:
- 「詐欺じゃないですか」
- 「難しくてわからない」
- 「家族に相談したい」

**対策**:
- 詐欺差別化3点セットを丁寧に説明
- 専門用語を使わない（「ウォレット」→「SBI証券のようなもの」）
- 家族も一緒に説明する機会を作る

---

### タイプC: 詐欺被害者

**特徴**: 極度の不信感
**頻出反論**:
- 「前に騙されたので信用できない」
- 「お金を預けるのが怖い」

**対策**:
- 詐欺差別化3点セットを最優先で伝える
- 小額テストを提案（「まず実力を見てください」）
- 急かさない（「信頼は実績でしか作れません」）

---

### タイプD: 資金制約あり

**特徴**: 「お金がない」と明言
**頻出反論**:
- 「今は余裕がない」
- 「もっと安くならないか」

**対策**:
- 小額テストを提案（10万円から）
- 無理を強要しない（「無理は禁物です」）
- 時期を延ばす（「次の機会にしましょう」）

---

### タイプE: 税務目的

**特徴**: 「計算だけしてほしい」
**頻出反論**:
- 「投資は興味ない」
- 「計算だけでいい」

**対策**:
- まず税金計算を完了させる（投資の話は一切しない）
- 実績を見せてから提案（「もしご興味があれば」）
- 完全に引く（信頼を守る）

---

### タイプF: 投資経験者

**特徴**: 専門用語を使う
**頻出反論**:
- 「他の投資と比較したい」
- 「利回りはどのくらいか」

**対策**:
- データと数字で説明
- 権威性を提示（「世界トップ10」「300億円」）
- 感情論不要（論理で攻める）

## 反論切り返しの5原則

### 原則1: 共感してから説明
```
✗ 「それは違います」
○ 「そう思われるのは当然です。その上で～」
```

### 原則2: 否定しない
```
✗ 「その考えは間違ってます」
○ 「なるほど、そういう見方もありますね」
```

### 原則3: 選択肢を出す
```
✗ 「これしかないです」
○ 「AとBがあります。推奨はAです」
```

### 原則4: 引く勇気を持つ
```
✗ 「なんとかなりませんか」
○ 「承知しました。また機会があれば」
```

### 原則5: 切り返し後に必ず確認
```
「どう思いますか？」
「これで少し安心していただけましたか？」
「何か他に気になることはありますか？」
```

## 切り返し成功率を上げる3つのコツ

### コツ1: 暗記ではなく理解
- フレーズを丸暗記するのではなく、「なぜこの切り返しが効くのか」を理解する
- 顧客の心理を読む（「なぜこの反論が出たのか」）

### コツ2: 声のトーンを柔らかくする
- 切り返しの言葉が正しくても、トーンが強いと逆効果
- 友達に話すような柔らかさで

### コツ3: 切り返し後に沈黙を作る
- 切り返したら3秒待つ
- 顧客が考える時間を与える
- 沈黙を埋めるために連続で話さない
