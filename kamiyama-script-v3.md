## 🟥 スクリプトを読む前に — お前の前提を壊せ

**お前の中にこういう前提がないか？**

> 「俺が正しいことを言えば、相手は理解するはず」

**これは間違い。**

お前の説明は正しい。内容も合ってる。でも正しいだけじゃ相手は動かない。

人は「正しい話」を聞いて動くんじゃない。**「自分に関係ある話」を聞いて動く。**

お前が100点の説明をしても、相手が「これ俺の話だ」と感じなければゼロ。伝わっていないのは、お前が間違っているからじゃない。**相手が「自分ごと」だと思っていないから。**

### じゃあどうすればいいのか

**説明を始める前に、一言だけ足せ。**

```
「○○さんに○○になって欲しいから、この話をします」
```

たったこれだけ。これがあるだけで、相手は「この人は俺のために話してくれてるんだな」と思う。聞く姿勢が変わる。

**例:**

```
✗ 今のお前の言い方:
「このファンドは年利○%で、○○の実績があって、リスク分散されていて…」
→ 相手の頭:「なんでこの話してるの？」「売り込みか？」

○ 一言足すだけ:
「○○さん、さっき老後が不安だとおっしゃってましたよね。
 正直それを聞いて、○○さんにはちゃんと安心できる状態になって欲しいと思いました。
 だから1つだけ大事な話をさせてください。」
→ 相手の頭:「俺のために言ってくれてるんだな。聞こう」
```

**同じ内容の説明でも、前提があるだけで相手の受け取り方が変わる。**

### 今日からやること（1つだけ）

説明を始める前に毎回、必ずこう言え:

**「○○さんに○○になって欲しいから、この話をします」**

これが言えないなら、その説明はお前のためにやっている。相手のためにやっていない。

## 🟧 演説モードに気づけ — 3つの自己チェック

お前が通話中に「演説モード」に入る原因は3つある。常に3つ同時ではなく、タイミングで変わる。だから**通話中に自分で気づく力**が必要。

### A: 知識を見せたいモード
**サイン:** 相手が聞いてないのに説明が止まらない。自分の話に酔っている。
**チェック:** 「この説明、相手に聞かれたか？」
**処方:** 聞かれてないなら止めろ。お前の知識披露はお前のためであって相手のためじゃない。

### B: 沈黙が怖いモード
**サイン:** 相手が黙った瞬間に話し始める。間が空くと不安になる。
**チェック:** 「今、沈黙を埋めようとしてないか？」
**処方:** 3秒数えろ。相手が先に話すまで待て。沈黙は敵じゃない。相手が考えている時間。

### C: 顧客に興味がないモード
**サイン:** 相手の名前・状況・前回の話を覚えていない。メモを見ていない。
**チェック:** 「この人の家族構成、趣味、悩みを言えるか？」
**処方:** 言えないなら興味を持て。興味がない相手に売れるわけがない。メモを読んでから電話しろ。

### 通話中の30秒ルール

**30秒以上連続で話したら、一度止まれ。**

止まって、相手に質問を投げろ:
- 「ここまでで何か気になることありますか？」
- 「○○さんはどう思いますか？」
- 「わかりにくいところありますか？」

30秒以上の連続トークは演説。演説は営業じゃない。

## この営業の本質

**売るな。顧客の言葉で出た悩みを、顧客自身の結論として解決させろ。**

顧客が「売られた」と感じた瞬間に負け。顧客が「自分で決めた」と感じれば勝ち。

### 成約の公式

```
【前半】 仲良くなる → 不安・夢・現状を引き出す
【中間】 ギャップを見せる（現実 vs 理想）
【後半】 投資 = そのギャップを埋める手段として接続
```

この順番を絶対に守れ。後半から入ったら終わり。

## 心理フロー: 顧客の5段階

```
① 警戒「誰だこいつ」
  ↓ 【仲良くなる】共感・雑談・問題解決
② 安心「悪い人じゃなさそう」
  ↓ 【ヒアリング】不安・夢・目標を引き出す
③ 気づき「このままだとまずいかも」
  ↓ 【ギャップ提示】現実と理想の差を見せる
④ 期待「解決できるかも」
  ↓ 【手段提示】投資をギャップの橋渡しとして提案
⑤ 決断「やってみよう」
  ↓ 【行動】小額テスト → 成功体験 → 次のステップ
```

**今の部下に足りないのは①→②と②→③の移行。** いきなり④から入るから失注する。

## STEP 0: 通話前準備（5分）

### やること
- 前回の通話メモを読む（約束事項・顧客の状況・家族構成・趣味）
- 顧客タイプを確認（下記）
- **前回の雑談で出た話題を1つ用意**（「お孫さんの運動会はどうでした？」等）

### 顧客タイプ判定

| タイプ | 見分け方 | 心理の動かし方 |
|---|---|---|
| A: 慎重型 | 質問が多い、即決しない | 論理+データ。時間を与える。圧は逆効果 |
| B: 高齢者 | 機械苦手、聞き返す | 1つずつ確認。専門用語禁止。安心感最優先 |
| C: 詐欺被害者 | 過去の被害を語る | まず被害解決。信頼は実績でしか作れない |
| D: 資金制約あり | 「お金がない」 | 少額のみ。高額を出した瞬間に終わる |
| E: 税務目的 | 「計算だけ」 | 税務を完了させてから。商品は絶対に後 |
| F: 投資経験者 | 専門用語を使う | データと数字。感情論不要 |
| G: 夢追求型 | 「○○したい」「いつかは」 | 夢を具体化→ギャップ→手段として投資 |

## STEP 1: 前半 —「仲良くなる」（警戒→安心→信頼）

**このSTEPの目標: 顧客に「この人は味方だ」と思わせる**

### 最初の3分: 掴み（警戒を解く）

```
「お忙しい中ありがとうございます。
 ○○から引き継ぎました、△△と申します。
 
 ○○からお客様の状況は共有いただいております。
 改めてご挨拶と、今後のことをお話しさせてください。」
```

**ここでやること:**
- 声のトーンを柔らかくする（友達と話すように）
- 顧客の名前を呼ぶ（「○○さん」）
- 焦らない。ゆっくり話す

**ここでやらないこと:**
- 自社サービスの説明
- 商品の話
- 「お時間大丈夫ですか？」（聞くと「忙しい」と言う口実を作る）

### 3〜10分: 雑談で壁を崩す

**雑談は営業の無駄ではない。信頼の土台。**

```
「○○さんはお住まいはどちらですか？」
「お仕事は何をされてるんですか？」
「休みの日は何されてますか？」
「ご家族はいらっしゃいますか？」
```

**雑談のルール:**
- 顧客7：自分3の割合で話す
- 相槌を入れる（「なるほど」「そうなんですね」「いいですね」）
- **共通点を見つけたら深掘りする**（「え、自分もゴルフやるんですよ！」）
- **メモを取る**（次回の通話で「先日のゴルフはどうでした？」と使う）

**神山が雑談で使うフレーズ:**
- 「気を使わずバチバチ言ってください」（対等な関係を作る）
- 「全然安心してください」（不安を和らげる）
- 「一緒に歩いていきましょう」（伴走者ポジションを取る）

### 10〜15分: 困りごとを聞く（問題解決で信頼を作る）

```
「改めてなんですけど、何かお困りのことやご不安なことはありますか？
 投資の話じゃなくても全然大丈夫です。」
```

**顧客が困りごとを言ったら → その場で解決する or 次回までに調べて報告**

例:
- 「ウォレットの操作がわからない」→ 画面共有で一緒に操作
- 「前の業者とトラブルがある」→ 対応方法を調べて報告
- 「税金の計算がわからない」→ 税金計算を対応する

**⚠️ ここが最重要。問題解決 = 信頼の最短ルート。**
問題を解決した人間の言うことは聞く。解決もしていない人間の投資提案は聞かない。

## STEP 2: ヒアリング — 不安・夢・現状を引き出す（安心→気づき）

**このSTEPの目標: 顧客の口から「不安」「夢」「現状の資産」を引き出す**

### なぜ引き出すのが先か

顧客自身が口に出した不安・夢だけが「本物」。
こちらが「お金困ってますよね？」と言っても刺さらない。
顧客が「正直、将来のお金が不安で…」と自分で言った時に初めてスイッチが入る。

### 不安を引き出す質問（パターン1: 不安解消型）

```
「お仕事は順調ですか？」
「将来のことで何か気になっていることはありますか？」
「老後の資金とか、考えたことありますか？」
「今のお給料で、10年後も安心できそうですか？」
```

**反応パターンと対応:**
- 「まあ、正直不安ですね…」→ 深掘り「具体的にどんなことが？」
- 「考えたくないですね笑」→ 軽く「ですよね笑 でも大事ですよね」
- 「別に困ってない」→ 無理に掘らない。パターン2（夢）に切り替え

### 夢を引き出す質問（パターン2: 夢実現型）

```
「○○さん、もしお金の心配が一切なかったら何したいですか？」
「どんな生活が理想ですか？」
「5年後、10年後、どうなっていたいですか？」
「お子さんにはどんな教育を受けさせたいですか？」
```

**この質問の目的: 顧客の中に「理想の未来」を作る。**
具体的であればあるほど良い。「海外旅行」より「ハワイのコンドミニアムに1ヶ月」の方が刺さる。

### 現状を確認する質問（パターン3: ギャップ型）

```
「今、貯蓄はどのくらいありますか？（答えにくければざっくりで）」
「毎月の余剰資金はどのくらいですか？」
「今何か投資はされていますか？」
「年金とか、将来の収入源は何かありますか？」
```

**ここで出た数字を必ずメモ。後で使う。**

### ヒアリングの禁止事項

- 尋問にしない。会話の流れで自然に聞く
- 答えたくなさそうなら引く（「無理に答えなくて大丈夫です」）
- **聞いたことを否定しない**（「それは少ないですね」はNG → 「なるほど、そうなんですね」）

## STEP 3: 中間 — ギャップを見せる（気づき→期待）

**このSTEPの目標: 顧客に「このままだとまずい」「何かしないと」と気づかせる**

### ギャップの作り方

STEP 2で引き出した「夢/目標」と「現状」を並べるだけ。

```
「○○さん、先ほど○○したいとおっしゃってましたよね。
 素敵な目標だと思います。
 
 ざっくりですけど、それを実現するには○○万円くらい必要になります。
 今のペースだと、正直なところ届くのは難しいかもしれません。
 
 …でも、方法がないわけじゃないんです。」
```

### パターン別ギャップ提示

**パターン1: 不安解消型**
```
「さっき将来のお金が不安だとおっしゃってましたよね。
 実際、今の日本では年金だけで老後を過ごすのは厳しいと言われています。
 
 ○○さんの場合、今のペースだと65歳時点で○○万円くらい。
 でも、少しずつ増やしていく方法はあります。」
```

**パターン2: 夢実現型**
```
「○○さんの夢、すごくいいですね。
 それを実現するには、ざっくり○○万円くらい必要です。
 
 今の収入と貯蓄だと、何もしなければ届かない。
 でも、資産を働かせれば届く距離にあります。」
```

**パターン3: 資産不足型**
```
「○○さんの目標に対して、今の資産では正直足りません。
 でも逆に言えば、足りない分を埋める手段があれば届くということです。
 
 その手段の一つが、投資です。」
```

### ギャップ提示のルール

- **顧客が自分で「何かしないとまずい」と思う前に提案しない**
- 焦らせない。煽らない。事実を並べるだけ
- 「どう思いますか？」と聞いて、顧客の反応を待つ
- 顧客が「でも…」と言ったら共感してから進む

## STEP 4: 後半 — 手段として投資を提示（期待→決断）

**このSTEPの目標: 投資を「顧客の悩みを解決する手段」として接続する**

### 接続の仕方

**NG: 「投資しませんか？」（売り込み）**
**OK: 「○○さんの○○を解決する方法として、こういうものがあります」（解決策の提示）**

```
「○○さんがおっしゃった○○、私も同じように大事だと思います。

 その目標に近づく方法の一つとして、
 弊社がサポートしている運用をご紹介させてください。
 
 もちろん強制ではありません。
 ご興味がなければ全く問題ないです。」
```

### 信頼構築: 詐欺差別化3点セット（必須）

```
「この業界、不誠実な業者が多いのは事実です。
 弊社が他と違う点を3つだけ。

 1. 入金先はお客様ご自身の正規口座。弊社に預けません。
 2. 報酬は成功報酬。利益が出なければ弊社の報酬はゼロ。
 3. 契約書は弁護士監修。全て書面に残ります。

 まず実力を見ていただいて、
 信頼できると思ったら次に進む。それで全く問題ありません。」
```

### 小額テストの提案

```
「まずは痛手にならない金額で、弊社の実力を見ていただけますか。
 ○○万円くらいから始められます。
 
 結果が出なかったらやめていただいて構いません。」
```

**金額の上限（絶対守れ）:**
| 可処分資金 | 提案上限 |
|---|---|
| 〜50万円 | 10万円以下 |
| 50〜200万円 | 30万円以下 |
| 200〜500万円 | 50万円以下 |
| 500万円超 | 100万円以下 |

### クロージング判断

| 顧客の反応 | 対応 |
|---|---|
| 前向き | 「では進めましょうか」 |
| 迷い中 | 「○日までにお考えください」 |
| 消極的 | 「無理にとは言いません。次の機会に」 |
| 明確な拒否 | 「承知しました」→ **再プッシュ禁止** |
| 「お金がない」 | 「無理は禁物です」→ 提案終了 |

## STEP 5: フォロー（信頼の維持 → 追加投資）

### 週次報告（必須・毎週同じ曜日・同じ時間）

```
「○○さん、お疲れ様です。今週のご報告です。

 現在の評価額: ○○万円（先週比+○○万円）
 利益率: ○○%

 何かご質問があればいつでもどうぞ。」
```

- **「順調です」だけは禁止。数字で語れ**
- 下がっている時こそ即報告。隠さない
- 通話は15分以内

### 暴落時（即日連絡必須）

```
「市場全体の動きです。○○に固有の問題ではありません。
 保有継続が最善と判断しています。
 ただし、○○さんのご判断を最優先にします。
 売却をご希望でしたら、すぐに対応します。」
```

### 追加投資提案の前提条件（全て満たすこと）

1. ✅ 初回ニーズが解決済み
2. ✅ テスト投資で利益が出ている
3. ✅ 顧客が満足している
4. ✅ 追加資金の余裕がある

**1つでも欠けていたら提案しない。**

### 追加投資の心理フロー

成功体験の直後に提案する（タイミングが鍵）。

```
「○○さん、前回○○万円の利益が出ましたね。おめでとうございます。
 
 次のステップとして、2つの選択肢があります。
 A: ○○万円で○○
 B: ○○万円で○○
 
 私の推奨はAです。理由は○○。
 もちろん、○○さんのペースで大丈夫です。」
```

## 仲良くなるための意識リスト

| 意識すること | 具体的な行動 |
|---|---|
| **名前を呼ぶ** | 「○○さん」を会話中に5回以上使う |
| **共通点を見つける** | 趣味・出身地・家族構成から1つ |
| **雑談メモを取る** | 次回「○○はどうでした？」と聞く |
| **相手の話を遮らない** | 最後まで聞いてから話す |
| **共感してから意見** | 「そうですよね」→「その上で」 |
| **否定しない** | 「それは違います」禁止 → 「なるほど、そういう考え方もありますよね」 |
| **自分の話もする** | 一方的に聞くだけだと尋問。自分の経験も少し出す |
| **声のトーンを合わせる** | 相手がゆっくりならゆっくり話す |

---

## ヒアリングの意識リスト

| 意識すること | 具体的な行動 |
|---|---|
| **質問は1つずつ** | 2つ同時に聞かない |
| **答えを急かさない** | 沈黙が3秒続いても待つ |
| **メモを取る** | 数字・固有名詞・感情ワードは必ず記録 |
| **深掘りする** | 「それってどういうことですか？」「具体的には？」 |
| **オウム返し** | 「将来が不安で…」→「将来が不安なんですね」 |
| **否定しない** | 何を言われても「なるほど」から入る |
| **尋問にしない** | 会話の流れで自然に。矢継ぎ早にしない |

## 規制リスク対応（v2から継承）

### 守るべきライン

| OK | NG |
|---|---|
| 顧客の質問に事実で回答 | 未公開情報で投資判断を誘導 |
| 正規取引所の取引サポート | 無登録の海外取引所へ誘導 |
| リスク許容度に合った提案 | 「借金してでも買え」 |
| 利益もリスクも正直に | 「絶対儲かる」「損はしない」 |
| 「税理士に確認してください」 | 「海外なら税金逃れできる」 |
| 契約書に基づくサービス | 口約束での運用開始 |
