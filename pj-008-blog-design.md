# PJ-008 Phase②: SEOブログ設計書

## サイト概要
- サイト名: 筋トレ科学ラボ（Kintore Science Lab）
- ドメイン: GitHub Pages（ken-fit-science.github.io）→ 無料
- フレームワーク: 静的サイト（Hugo or Jekyll）→ Sonnetが生成・デプロイ
- ジャンル: フィットネス（Threadsと統一）
- 収益: 楽天アフィリエイト（既存リンク流用）

## サイト構成

```
/
├── index.html          （トップ：最新記事一覧）
├── /articles/
│   ├── protein-guide/  （プロテイン完全ガイド）← 柱記事
│   ├── dumbbell-guide/ （ダンベル完全ガイド）← 柱記事
│   ├── beginner-3months/（初心者3ヶ月プログラム）← 柱記事
│   └── [個別記事...]
├── /about/             （プロフィール）
└── /sitemap.xml
```

## 記事戦略

### 柱記事（3本・各3000-5000字）— SEOメイン
1. **「プロテイン おすすめ 2026」** — WPC/WPI比較、コスパランキング、楽天リンク
2. **「可変式ダンベル おすすめ」** — 自宅トレの器具選び、楽天リンク
3. **「筋トレ 初心者 メニュー」** — BIG3解説、週2回プログラム

### 量産記事（週2本・各1000-2000字）
- Threadsのストック（pj-005-content-stock.md）を記事に展開
- 例: K-001「腹筋の真実」→ 1500字の記事に拡張

### SEOキーワード戦略
- ロングテール狙い（競合が弱い）
- 「プロテイン WPI おすすめ 安い」
- 「可変式ダンベル 初心者 おすすめ」
- 「筋トレ 週2回 効果」
- 「筋トレ 初心者 メニュー 自宅」

## Threads→ブログの導線
- Threads投稿の末尾に「詳しくはプロフィールのリンクから」
- ThreadsプロフィールにブログURLを設定
- ブログ記事の末尾にThreadsフォロー誘導

## cronスケジュール
| タスク | 頻度 | 実行者 |
|--------|------|--------|
| 柱記事生成 | ワンショット（初回3本） | Sonnet |
| 量産記事生成 | 週2回（水・土 21:00） | Sonnet |
| Threadsプロフィール更新 | ワンショット | Sonnet |
| ブログKPI（Search Console等） | 週次 | Sonnet |
