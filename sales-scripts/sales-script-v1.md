# 営業トークスクリプト v1
**作成日**: 2026-04-03
**対象**: 新人営業（初回営業電話用）
**目的**: 投資信託商品の初回アポイント取得
**ベース**: kamiyama-script-v3.mdをシンプル化

---

## 📋 使い方
- **通話前**: このスクリプトを1回音読して流れを頭に入れる
- **通話中**: 30秒以上連続で話したら必ず質問を挟む
- **通話後**: 「どこで詰まったか」をメモして次に活かす

---

## 🎯 この営業の目的
**アポイント取得 = 詳細説明の時間を30分もらう**

成約は後。今日のゴールは「会う約束」だけ。

---

## ステップ1: 掴み（最初の30秒）

### 目的
電話を切られないこと。「話を聞いてもいいかな」と思わせる。

### トーク例
```
お世話になります。○○（会社名）の○○と申します。
突然のお電話失礼します。

今日は、○○さんに将来の資産についてお伝えしたいことがあってお電話しました。
今、2〜3分だけお時間大丈夫でしょうか？
```

### ポイント
- 「売り込みです」と言わない。でも嘘もつかない
- 「2〜3分」と短く区切る → 相手が「まあ聞くだけなら」と思う
- 断られたら「またの機会に」と言って引く。しつこくしない

### よくある反応と対処
**「忙しいんですけど」**
→ 「承知しました。またの機会にさせてください。失礼します」（引く）

**「どういうご用件ですか？」**
→ 「○○さんの将来の資産についてお伝えしたいことがありまして」（目的を言う）

**「結構です」**
→ 「承知しました。失礼します」（引く）

---

## ステップ2: 雑談（信頼構築 1〜2分）

### 目的
「この人は売り込みだけじゃない」と思わせる。警戒を解く。

### トーク例
```
ありがとうございます。
ところで○○さん、今日はお休みですか？（または: お仕事中ですか？）

[相手の返答を聞く]

実は私も○○でして…（共通点を探す）
```

### ポイント
- **必ず相手に質問する**。自分の話ばかりしない
- 天気・地域・仕事・趣味など、軽い話題
- 相手が話したことを覚えておく（メモ必須）

### 雑談の終わり方
```
○○さん、実は今日お電話したのは…（ステップ3へ）
```

---

## ステップ3: ヒアリング（顧客の不安・夢を引き出す）

### 目的
相手の悩みを聞き出す。「この人は俺のことを理解してくれる」と思わせる。

### 質問例（順番に聞く）
1. **現状確認**
   ```
   ○○さん、将来のお金のことって考えたりしますか？
   ```

2. **不安の掘り下げ**
   ```
   何か不安に思っていることってありますか？
   （例: 老後、年金、子供の教育費、住宅ローン）
   ```

3. **理想の確認**
   ```
   理想を言うと、将来どんな状態になっていたいですか？
   ```

### ポイント
- **相手の言葉をそのまま復唱する**
  - 例: 「老後が不安なんですね」「お子さんの教育費が心配なんですね」
- 沈黙を恐れない。相手が考える時間を作る
- メモを取る（後で使う）

---

## ステップ4: ギャップ提示（問題を見せる）

### 目的
「このままだとマズいかも」と思わせる。

### トーク例
```
○○さん、さっき「老後が不安」っておっしゃってましたよね。

実は今、年金だけで生活するのって結構厳しくて、
一般的には老後資金2,000万円が必要って言われてるんです。

○○さんは今、老後のために何か準備されてますか？
```

### ポイント
- **相手が言った言葉を使う**（ステップ3で聞いた内容）
- 「売り込み」ではなく「事実」を伝える
- 「○○さんはどうですか？」と必ず聞く

---

## ステップ5: 接続（商品につなげる）

### 目的
「ちゃんと解決策がある」と思わせる。でも詳細は言わない。

### トーク例
```
○○さん、実は今回お電話したのは、
○○さんみたいに将来のお金のことを考えている方に、
ちゃんと準備する方法をお伝えしたくて。

ただ、電話でお伝えするには内容が多くて…
もし○○さんがご興味あれば、30分だけお時間いただいて、
詳しくお話しさせていただけませんか？
```

### ポイント
- **商品名は出さない**（詳細は会ってから）
- 「30分だけ」と短く区切る
- 「興味あれば」と押し付けない

---

## ステップ6: クロージング（アポ取得）

### 目的
日時を決める。

### トーク例
```
ありがとうございます。
それでは、○○さんのご都合がいい日はいつですか？

[日程候補を聞く]

○月○日の○時はいかがですか？

[確定]

それでは、○月○日○時にお電話させていただきます。
○○さん、今日はありがとうございました。
```

### ポイント
- **具体的な日時を決める**（「またで」は逃げ）
- 相手の都合を優先する
- 最後に感謝を伝える

---

## よくある反論と切り返し

### 「今は必要ないです」
```
承知しました。でも将来のことを考えるきっかけだけでも
お伝えできればと思うんです。
30分だけでも、お時間いただけませんか？
```

### 「お金がありません」
```
○○さん、実は少額から始められる方法もあるんです。
詳しくは一度お話しさせていただけませんか？
```

### 「他で契約してます」
```
そうなんですね。ちなみに、今の契約で満足されてますか？
もし何か気になることがあれば、セカンドオピニオンとして
お話しさせていただけませんか？
```

### 「忙しいです」
```
承知しました。では、○○さんのご都合がいい時間帯を
教えていただけますか？夜でも土日でも大丈夫です。
```

---

## 絶対にやってはいけないこと（NGリスト）

1. **30秒以上連続で話す** → 演説になる。必ず質問を挟め
2. **相手の話を遮る** → 最後まで聞け。話を奪うな
3. **専門用語を使う** → 相手はわからない。小学生でもわかる言葉で
4. **「でも」「しかし」で否定する** → 「なるほど」「そうですよね」で受け止めろ
5. **しつこくする** → 断られたら引く。次に繋げる

---

## 通話後にやること（必須）

1. **顧客情報をメモに記録**
   - 名前、年齢、家族構成
   - 不安・夢（相手が言った言葉そのまま）
   - 次回アポ日時

2. **自己評価（3項目のみ）**
   - どこで詰まったか？
   - どこがうまくいったか？
   - 次回どうするか？

3. **次回の準備**
   - 今回聞いた内容をもとに、次の提案を考える

---

## 成功の定義

**✅ アポイント取得 = 成功**
**❌ 断られた = 次に活かす**

断られることは失敗じゃない。断られ方から学べ。
100件電話して10件アポが取れれば十分。

---

## まとめ（1分で復習）

1. **掴み**: 2〜3分だけお時間ください
2. **雑談**: 相手に質問して警戒を解く
3. **ヒアリング**: 不安・夢を引き出す
4. **ギャップ**: このままだとマズいと思わせる
5. **接続**: 解決策があると伝える（詳細は言わない）
6. **クロージング**: 30分だけ時間をください

**この流れを守れば、アポは取れる。**
