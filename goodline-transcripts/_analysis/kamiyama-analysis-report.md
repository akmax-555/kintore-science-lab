# 神山 営業技術分析レポート
**分析日**: 2026年3月18日
**対象**: 成約顧客19名（18名分析完了）/ 通話282件 / 総通話時間252時間

---

## 1. 神山の営業の全体像

### 成約までの平均プロセス
- 平均通話回数: 19.2回
- 平均所要期間: 2-3ヶ月
- 1通話の平均時間: 約50分
- 成約率: 6.1%

### 営業の本質
神山の営業は**「伴走型」**。売り込まない。問題解決から入り、信頼を積み上げ、段階的に投資額を引き上げる。

---

## 2. 神山の5つの武器（共通パターン）

### 武器①: 入口は「問題解決」
- ウォレット復旧、取引所操作サポート、前業者トラブル対応
- **絶対に初回で商品を売らない**
- 「お困りごとを解決する人」として登場し、信頼の土台を作る

**再現フレーズ:**
> 「まずは状況を確認させてください」
> 「ご不安な点、何でも聞いてください」

### 武器②: 詐欺との差別化3点セット
全成約顧客に共通して使用。詐欺被害者が多い業界で最重要の信頼構築手段。

| 差別化ポイント | 説明フレーズ |
|---|---|
| ① 入金先 | 「入金先はお客様ご自身の正規口座です。弊社にお金を預けるのではありません」 |
| ② 報酬体系 | 「成功報酬2%。利益が出なければ弊社の報酬はゼロです」 |
| ③ 実績証明 | 「2000人以上の実績。契約書は弁護士監修です」 |

### 武器③: 段階的コミットメント
```
小額テスト（20-30万円）
  ↓ 成功体験の共有
中規模投資（100-300万円）
  ↓ 定期報告で信頼強化
大型投資（500万-1000万円超）
  ↓ 「次のラインを目指しましょう」
継続的な投資関係
```

- 各段階で「成功体験」を必ず挟む
- 成功体験の直後に次の提案を出す（タイミングが鍵）
- **断られたら即座に引く**（470万円提案を断られ→「また機会があれば」とサラリと引く）

### 武器④: 顧客タイプ別アプローチの使い分け

| タイプ | 特徴 | 神山のアプローチ |
|---|---|---|
| **慎重派・詐欺被害者** | 警戒心強い、見極めたい | 詐欺差別化3点→小額テスト→実力証明 |
| **投資経験者** | 論理・データ重視 | 市場分析共有→数字で証明→権威引用 |
| **IT初心者** | 操作が不安 | 画面共有で手取り足取り→依存関係形成 |
| **不安型** | 感情的に揺れる | 傾聴→話題分解→具体的数字→定期フォロー |
| **前業者不満型** | 他社へ不信感 | 「味方です」宣言→解約支援→対比で優位性 |
| **短期利益追求型** | 即結果を求める | 緊急性演出→希少性→即行動 |

### 武器⑤: 信頼構築の7つの柱

1. **長時間技術サポート**: 1-2時間の画面共有で手取り足取り
2. **個人的な話題の共有**: 家族・趣味・体調・週末の予定
3. **権威性の提示**: 「世界トップ10」「300億円」「分析チームが精査」
4. **リスクの開示**: 「上場後は下がることもある」（隠さないことで信頼獲得）
5. **承認技術**: 「素晴らしい行動力」「5年の投資歴、ベテランですね」
6. **定期フォロー**: 週1回、数字で報告。「木曜11時に連絡します」
7. **社会的証明**: 「他のお客様も同じように動いています」

## 3. フェーズ別・神山の型

### フェーズ1: 初回通話（最初の3分）

**パターンA: 新規顧客**
```
①「お忙しい中ありがとうございます」（敬意）
②「上司の神山と申します」（権威付け）
③「○○から経緯を共有させていただきました」（情報把握済みアピール）
④「まずは状況をお聞かせください」（顧客を主役にする）
```

**パターンB: 担当変更**
```
①「部署異動の関係で」（ポジティブな理由）
②「新規事業の展開でポジティブな話」（不安の払拭）
③「普段お客様と直接話すことはほぼない」（特別感）
④「今まで以上に手厚いサポートを」（格上げ感）
```

**パターンC: 長期空白後の再開**
```
①「ご無沙汰しております」（丁寧な再開）
②「お仕事落ち着きましたか？」（状況への共感）
③「市場が変わったので改めて精査しました」（空白をポジティブに）
④「少しずつでも進めていければ」（焦らせない）
```

### フェーズ2: 関係構築（2-5回目）

- 画面共有で操作サポート（口座開設・ウォレット設定）
- 顧客の投資経験・知識レベルを把握
- 家族構成・目標を聞き出す
- 小額テストの提案
- **次回予約を必ず確定**（「来週の○曜日、○時はいかがですか」）

### フェーズ3: 説明→納得→理解

**神山の説明の型:**
```
結論（1行）
 ↓
理由（箇条書き）
 ↓
具体例（数字・事例）
 ↓
顧客の利益（「○○様の場合は～」）
```

**複雑な話題の分解術:**
> 「2つに分けて話しましょう。まず○○の話、次に△△の話」

**専門用語の平易化:**
| 専門用語 | 神山の言い換え |
|---|---|
| ウォレット | SBI証券のようなもの |
| NFT | ヤフオク・メルカリのデジタル版 |
| エアドロップ | 株の配当のようなもの |
| トランザクション | 取引履歴 |

### フェーズ4: クロージング

**神山のクロージング5原則:**

1. **押し売りしない**: 「無理にとは言いません」「結果次第で判断してください」
2. **選択肢を出す**: 「AとBがあります。私の推奨はAです」
3. **希少性を演出**: 「今月限り」「96%埋まっています」「年2-3回の機会」
4. **当事者性を出す**: 「私も同じ方法で買っています」「個人でも持ちたい」
5. **次回を確定する**: 通話の最後に必ず次回日時を決める

**クロージングフレーズ集:**
> 「まずは実力を見てください」
> 「結果が出なかったらやめていただいて構いません」
> 「今の市場は仕込み時です」
> 「他のお客様も同じタイミングで動いています」
> 「決めちゃっていいんじゃないですか」（背中を軽く押す）

### フェーズ5: アフターフォロー

- 週1回の定期連絡（数字で報告）
- 価格変動時の即連絡（暴落時は特に重要）
- 次の投資機会の紹介
- 紹介の依頼（「いいものであれば近しい方にも」）

## 5. 問題点と改善案

### 部下が真似すべきでないもの

| 問題 | 影響 | 改善案 |
|---|---|---|
| 技術サポートが長すぎる（1-2時間） | 時間効率が悪い | 操作マニュアル動画を作成 |
| 同じ説明の繰り返し | 顧客を疲弊させる | 説明の「型」を作り簡潔化 |
| 説明が長い（30分超の理論説明） | 顧客の集中力が切れる | 要点を5分にまとめる |
| 「どうしますか？」と判断丸投げ | 顧客が迷う | 「AとBがある。推奨はA」 |
| 「確認します」で即答しない | 信頼性が下がる | 可能な限り即答 |
| 高額商品の唐突な提示 | 押し売り感 | 段階的に引き上げる |
| 「うんうんうん」の多用 | 軽薄な印象 | 相槌のバリエーションを増やす |

## 7. 結論: 神山の強みと再現性

### 強み（3つに集約）
1. **伴走型関係構築**: 問題解決→信頼→段階的投資。時間をかけるが成約後のLTVが高い
2. **顧客適応力**: 6タイプの顧客に合わせてアプローチを変える柔軟性
3. **リスク開示で信頼獲得**: 隠さないことが最大の差別化

### 再現可能なもの（部下に落とし込めるもの）
- 初回3分の型（3パターン）
- 詐欺差別化3点セット
- 説明の型（結論→理由→具体例→顧客の利益）
- 段階的コミットメントの設計
- クロージング5原則
- 定期フォロー体制

### 再現困難なもの（神山の個人スキル）
- 長時間通話に耐える忍耐力
- 顧客の感情を声のトーンで読み取る力
- バラバラに見える会話を最終的に着地させる力
- 10ヶ月伴走する粘り強さ
