# 顧客ヒアリングシート（お見合いシート型）
**作成日**: 2026-03-26
**目的**: これを全部埋めたら「○○さんはこういう人です」と説明できる状態を作る
**使い方**: 初回〜3回目の通話で埋める。一度に全部聞くな。会話の流れで自然に。

---

## 📋 基本情報（初回で埋める）

| 項目 | 記入欄 | なぜ聞くか |
|---|---|---|
| 名前 | | 呼び方の確認。「○○さん」を使う |
| 年齢 | | 提案内容の調整（20代と60代では全く違う） |
| 住まい（都道府県） | | 雑談の材料。対面可能かの判断 |
| 電話がつながりやすい時間帯 | | 連絡のタイミング。約束を守るための情報 |
| 呼ばれたい呼び方 | | 距離感の調整。苗字か名前か |

## 📈 投資経験・過去の経歴（2回目以降）

| 項目 | 記入欄 | なぜ聞くか |
|---|---|---|
| 投資経験の有無 | | アプローチの調整 |
| 何に投資したことがあるか | | 知識レベルの把握 |
| 投資で儲かった経験 | | 成功体験があると前向きになりやすい |
| 投資で損した経験 | | 地雷を踏まないため |
| 過去に詐欺被害はあるか | | 最重要。信頼構築のアプローチが変わる |
| 被害金額（あれば） | | 心理的ダメージの深さの把握 |
| 他社のサービスを使っているか | | 競合との差別化 |
| 仮想通貨の知識レベル（1〜5） | | 説明の深さを調整 |
| ウォレット・取引所の有無 | | 技術サポートの必要度 |

## 🏷️ 顧客タイプ判定（営業側が記入）

| 項目 | 記入欄 |
|---|---|
| タイプ（A〜G） | |
| 判定理由 | |
| アプローチ方針 | |
| 注意事項 | |
| 絶対やってはいけないこと | |

**タイプ早見表:**
- A: 慎重型 → 論理+データ。圧は逆効果
- B: 高齢者 → 専門用語禁止。安心感最優先
- C: 詐欺被害者 → まず被害解決。信頼は実績でしか作れない
- D: 資金制約あり → 少額のみ
- E: 税務目的 → 税務完了が先
- F: 投資経験者 → データと数字
- G: 夢追求型 → 夢を具体化→ギャップ→投資

## 聞く順番ガイド

| タイミング | 聞くこと | 注意 |
|---|---|---|
| **初回の雑談** | 基本情報 + 趣味・休日 + 家族 | 尋問にしない。会話で自然に |
| **初回の後半** | 困りごと | 「投資の話じゃなくても大丈夫です」 |
| **2回目** | 仕事の詳細 + 投資経験 + 不安 | 前回のメモを見てから電話 |
| **3回目** | お金の状況 + 夢・理想 | 信頼ができてから。無理に聞かない |
| **毎回** | 通話記録 + 雑談メモ | 5分以内に記入。後から書くと忘れる |
