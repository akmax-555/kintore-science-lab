# 営業スクリプトv3 — セルフチェックリスト

**対象**: 部下4人（新人〜中堅営業）
**用途**: 通話前・通話中・通話後にこのリストを見てチェック
**更新日**: 2026-03-29

---

## ⚠️ 最優先: 絶対禁止リスト（破った時点でアウト）

通話前に音読しろ。どれか1つでも破ったらその通話は失敗。

| # | 禁止行動 |
|---|---|
| 1 | 「絶対」「確実」「必ず儲かる」と言う |
| 2 | 顧客が拒否したのに再プッシュする |
| 3 | 1時間を超える通話をする |
| 4 | 「借金してでも」「融資で投資しろ」と言う |
| 5 | 税務相談から強引に商品転換する |
| 6 | 「海外なら税務署に追われない」と言う |
| 7 | 「なんでですか？」と詰める |
| 8 | 感情的拒否を論理で覆そうとする |
| 9 | 約束した時間に連絡しない（5分のズレでも失注） |
| 10 | 感情的に対応する |

## 🔵 通話前チェックリスト（5分）

### A: 準備は整っているか？

- [ ] 前回の通話メモを読んだか
- [ ] 顧客の家族構成・趣味・悩みを言えるか
- [ ] 前回の約束事項を確認したか
- [ ] 前回の雑談で出た話題を1つ用意したか（例: 「お孫さんの運動会どうでしたか？」）
- [ ] 顧客タイプを確認したか（慎重型/高齢者/詐欺被害者/資金制約あり/税務目的/投資経験者/夢追求型）
- [ ] 通話時間の上限を決めたか（初回30分、フォロー15分）

### B: 心構えは整っているか？

- [ ] 「正しい説明をすれば伝わる」という前提を捨てたか
- [ ] 「売る」のではなく「顧客の悩みを解決する」マインドになっているか
- [ ] 今日の通話の目標を1つ決めたか（例: 「仲良くなる」「不安を引き出す」「次回予約を取る」）
- [ ] 沈黙を恐れていないか（3秒待つ練習をしたか）

## 🟢 初回通話チェックリスト（30分以内）

### STEP 0: 掴み（0-3分）— 警戒を解く

- [ ] 声のトーンを柔らかくしているか（友達と話すように）
- [ ] 顧客の名前を呼んだか（「○○さん」）
- [ ] 焦っていないか（ゆっくり話しているか）
- [ ] 自社サービスの説明をしていないか（したらNG）
- [ ] 商品の話をしていないか（したらNG）
- [ ] 「お時間大丈夫ですか？」と聞いていないか（聞くとNG）

### STEP 1: 仲良くなる（3-15分）— 安心→信頼

#### 雑談（3-10分）

- [ ] 顧客7：自分3の割合で話しているか
- [ ] 相槌を入れているか（「なるほど」「そうなんですね」「いいですね」）
- [ ] 共通点を見つけたか（1つ以上）
- [ ] 共通点を見つけたら深掘りしたか（「え、自分もゴルフやるんですよ！」）
- [ ] 雑談メモを取ったか（次回の通話で使う）
- [ ] 顧客の名前を会話中に5回以上使ったか

#### 困りごとを聞く（10-15分）

- [ ] 「お困りのことやご不安なことはありますか？」と聞いたか
- [ ] 「投資の話じゃなくても全然大丈夫です」と伝えたか
- [ ] 顧客が困りごとを言ったか
- [ ] 困りごとに対して解決アクションを取ったか（例: 画面共有で操作、次回までに調べる、税金計算を対応する）

### STEP 2: ヒアリング（15-25分）— 不安・夢・現状を引き出す

#### 質問の仕方

- [ ] 質問は1つずつ投げているか（2つ同時に聞いていないか）
- [ ] 答えを急かしていないか（沈黙が3秒続いても待ったか）
- [ ] 尋問になっていないか（会話の流れで自然に聞いているか）

#### 引き出せたか

- [ ] 不安 or 夢 or 現状の資産のいずれか1つ以上を引き出せたか
- [ ] 顧客が自分の言葉で不安/夢を語ったか
- [ ] 数字を引き出せたか（貯蓄・余剰資金・投資額）
- [ ] メモを取ったか（数字・固有名詞・感情ワードは必ず記録）

#### 対応

- [ ] 顧客が何を言っても「なるほど」から入ったか
- [ ] オウム返しをしたか（「将来が不安で…」→「将来が不安なんですね」）
- [ ] 顧客の答えを否定していないか（「それは少ないですね」はNG）
- [ ] 答えたくなさそうなら引いたか（「無理に答えなくて大丈夫です」）

### STEP 3: 終了（25-30分）

- [ ] 次回予約を取ったか（日時を確定したか）
- [ ] 30分以内に終わったか
- [ ] **投資の話をしていないか（したらNG）**

## 🟡 2回目以降の通話チェックリスト（30分以内）

### STEP 0: 振り返り（0-5分）

- [ ] 前回の雑談メモから話題を振ったか（「○○はどうでした？」）
- [ ] 前回の依頼事項を報告したか（問題解決の結果を伝えたか）

### STEP 3: ギャップ提示（5-15分）— 気づきを作る

#### 準備

- [ ] 顧客の言葉をそのまま使ったか（「○○さんがおっしゃった○○」）
- [ ] STEP 2で引き出した「夢/目標」と「現状」を並べたか

#### 提示の仕方

- [ ] 押し売りではなく事実を並べたか
- [ ] 焦らせていないか、煽っていないか
- [ ] 「どう思いますか？」と聞いて顧客の反応を待ったか
- [ ] 顧客が「でも…」と言ったら共感してから進んだか

#### 投資への接続（初めて投資の話をする場合のみ）

- [ ] 「○○さんに○○になって欲しいから、この話をします」と前置きしたか
- [ ] 投資を「顧客の悩みを解決する手段」として接続したか
- [ ] 「興味がなければ全く問題ないです」と伝えたか

### STEP 4: クロージング（15-25分）— 決断を促す

#### 信頼構築

- [ ] 詐欺差別化3点を伝えたか（入金先・成功報酬・契約書）
- [ ] 小額テストを提案したか（痛手にならない金額）
- [ ] 可処分資金の20%以下で提案したか

#### 判断

- [ ] 顧客の反応を確認したか
- [ ] 前向き → 「では進めましょうか」
- [ ] 迷い中 → 「○日までにお考えください」
- [ ] 消極的 → 「無理にとは言いません」
- [ ] 明確な拒否 → 「承知しました」**→ 再プッシュしていないか（したらNG）**
- [ ] 「お金がない」→ 「無理は禁物です」→ 提案終了したか

### STEP 5: 終了（25-30分）

- [ ] 次回予約を確定したか
- [ ] 30分以内に終わったか

## 🟠 フォロー通話チェックリスト（週1回・15分以内）

### 報告

- [ ] 週次報告を毎週同じ曜日・同じ時間に実施しているか
- [ ] 数字で報告したか（評価額・先週比・利益率）
- [ ] 「順調です」だけで済ませていないか（したらNG）
- [ ] 下がっている時こそ即報告したか（隠していないか）

### 暴落時（即日連絡必須）

- [ ] 市場全体の動きであることを伝えたか
- [ ] 保有継続の判断理由を伝えたか
- [ ] 顧客の判断を最優先にすることを伝えたか
- [ ] 売却希望に即対応する姿勢を伝えたか

### 追加投資提案（前提条件を全て満たすこと）

- [ ] 初回ニーズが解決済みか
- [ ] テスト投資で利益が出ているか
- [ ] 顧客が満足しているか
- [ ] 追加資金の余裕があるか
- [ ] 成功体験の直後に提案しているか

## 🔴 通話中リアルタイムチェック（30秒ルール）

### 演説モードに気づけ

**30秒以上連続で話したら、一度止まれ。**

- [ ] 相手が聞いてないのに説明が止まらないか（知識を見せたいモード）
- [ ] 相手が黙った瞬間に話し始めていないか（沈黙が怖いモード）
- [ ] 相手の名前・状況・前回の話を覚えているか（顧客に興味があるか）

### 止まったら質問を投げろ

- 「ここまでで何か気になることありますか？」
- 「○○さんはどう思いますか？」
- 「わかりにくいところありますか？」

### 次回予約

- [ ] 次回の日時を確定したか

- [ ] 絶対禁止リスト10項目のいずれかを破っていないか

## 💡 成約率を上げるための意識リスト

### 仲良くなる8項目

- [ ] 名前を呼ぶ（会話中に5回以上）
- [ ] 共通点を見つける（趣味・出身地・家族構成から1つ）
- [ ] 雑談メモを取る（次回「○○はどうでした？」と聞く）
- [ ] 相手の話を遮らない（最後まで聞いてから話す）
- [ ] 共感してから意見（「そうですよね」→「その上で」）
- [ ] 否定しない（「それは違います」禁止 → 「なるほど、そういう考え方もありますよね」）
- [ ] 自分の話もする（一方的に聞くだけだと尋問）
- [ ] 声のトーンを合わせる（相手がゆっくりならゆっくり話す）

### ヒアリング7項目

- [ ] 質問は1つずつ（2つ同時に聞かない）
- [ ] 答えを急かさない（沈黙が3秒続いても待つ）
- [ ] メモを取る（数字・固有名詞・感情ワード）
- [ ] 深掘りする（「それってどういうことですか？」「具体的には？」）
- [ ] オウム返し（「将来が不安で…」→「将来が不安なんですね」）
- [ ] 否定しない（何を言われても「なるほど」から入る）
- [ ] 尋問にしない（会話の流れで自然に）
