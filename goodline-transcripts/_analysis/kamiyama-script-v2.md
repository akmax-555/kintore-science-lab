# 営業トークスクリプト v2
**更新日**: 2026-03-18
**ベース**: 成約18件の勝ちパターン + 未成約53件の失注分析 + 金融庁WG規制資料14本
**対象**: 新人〜中堅営業
**v1からの変更点**: 失注5大パターンの防止策を全STEPに組み込み、規制リスク対応を追加

## 🔴 絶対禁止リスト（破った時点でアウト）

| # | 禁止行動 | 理由（実データ） |
|---|---|---|
| 1 | **「絶対」「確実」「必ず儲かる」** | 金商法上の断定的判断の提供。刑事罰対象 |
| 2 | **顧客の拒否を無視して再プッシュ** | 失注率50%の直接原因。1回断られたら即引け |
| 3 | **1時間超の通話** | 失注率46%。顧客は疲弊して判断不能→保留→離脱 |
| 4 | **「借金してでも買え」「融資で投資しろ」** | 顧客適合性義務違反。金融庁が明確に問題視 |
| 5 | **税務相談からの強引な商品転換** | 失注率58%の最大原因。ニーズを先に解決しろ |
| 6 | **「海外なら税務署に追われない」** | 脱税教唆。刑事リスク |
| 7 | **「なんでですか？」「はっきり言ってください」** | 圧迫面接。顧客の信頼を即座に破壊 |
| 8 | **顧客が「感覚的に嫌」と言ったのに無視** | 感情的拒否は論理で覆せない。引け |
| 9 | **約束した時間に連絡しない** | 信頼崩壊。5分のズレでも失注した事例あり |
| 10 | **感情的対応（「ショック」「投げやり」等）** | プロ失格。顧客の判断を尊重しろ |

## STEP 0: 通話前準備（5分）

### 顧客カルテ確認
通話前に以下を必ず確認:
- 前回の通話内容・約束事項
- 顧客の目標金額・財務状況
- 顧客タイプ（下記参照）
- 未解決の問い合わせ・依頼

### 顧客タイプ判定（初回3分で分類）

| タイプ | 見分け方 | 対応方針 |
|---|---|---|
| **A: 慎重型** | 質問が多い、即決しない、「考えます」 | 論理+データ。時間を与える。圧NG |
| **B: 高齢者** | 機械苦手、聞き返しが多い | 簡潔に。専門用語禁止。対面推奨 |
| **C: 詐欺被害者** | 過去の被害を語る、警戒心極強 | まず被害解決。投資提案は実績後 |
| **D: 資金制約あり** | 「お金がない」「ロック中」「生活費で…」 | 少額プランのみ。高額提案は絶対NG |
| **E: 税務目的** | 「計算だけ」「申告が…」 | 税務を完了させてから。商品は後 |
| **F: 投資経験者** | 専門用語を使う、比較検討する | データと数字で勝負。感情論不要 |

**⚠️ タイプD・Eは特に注意。** これらの顧客に商品を押し込むと、失注率80%超。

## STEP 1: 初回通話（目標: 次回予約を取る）

### ルール
- **最大30分**。超えたら「次回に続きをやりましょう」で切る
- **初回で商品を売らない**。売るのは「次の約束」だけ
- **顧客の本来ニーズを特定する**（税務？復旧？運用？情報？）

### 開始〜3分: 掴み

```
「お忙しい中ありがとうございます。
 ○○（前担当）から引き継ぎました、△△と申します。
 
 ○○からお客様の状況は共有いただいております。
 改めてご挨拶と、今後のサポート体制についてお話しさせてください。
 
 まず、何かお困りのことやご不安なことはありますか？」
```

**やること:**
- 最初の3分は**聞く**。話さない
- 「なるほど、それは心配ですよね」と共感
- メモを取る（顧客のニーズ・不安・状況）

**やらないこと:**
- 「はいはいはい」の相槌連打
- 自社サービスの説明を始める
- 顧客の話を遮る

### 3分〜10分: ニーズ特定 + 信頼構築

**まずニーズを確認する。これが最優先。**

```
「お話を伺うと、○○様のご要望は[税金計算 / 復旧 / 運用 等]
 ということでよろしいでしょうか？

 承知しました。まずはそちらをしっかり対応させていただきます。」
```

**⚠️ 顧客が「税金計算だけ」「復旧だけ」と言ったら、それだけやれ。**
運用提案は、その依頼を完了してから。順序を逆にした瞬間に失注する。

次に信頼構築（詐欺差別化3点セット）:

```
「この業界、不誠実な業者が多いのは事実です。
 弊社が他と違う点を3つだけ。

 1. 入金先はお客様ご自身の正規口座。弊社に預けません。
 2. 報酬は成功報酬。利益が出なければ弊社の報酬はゼロ。
 3. 契約書は弁護士監修。全て書面に残ります。

 まず実力を見ていただいて、
 信頼できると思ったら次に進む。それで全く問題ありません。」
```

### 10分〜25分: 状況ヒアリング

```
「いくつか確認させてください。

 ・現在、仮想通貨の保有はありますか？
 ・これまでの投資経験はどのようなものですか？
 ・何か目標とされている金額はありますか？
 ・ご予算として、動かせる範囲はどのくらいですか？」
```

**🔴 財務状況チェック（必須）:**
- 「動かせるお金がない」→ 少額プランのみ提案。高額は絶対出すな
- 「借金がある」「ローン返済中」→ 投資提案自体を控える
- 「生活費で余裕がない」→ 税務サポートに絞る。投資はNG

### 25〜30分: 次回予約

```
「ありがとうございます。
 ○○様の状況に合わせて最適なプランを準備します。

 来週の○曜日、○時あたりはいかがですか？
 30分もあれば十分です。」
```

**次回予約を取らずに終了するのは禁止。**

## STEP 2: 2-3回目（目標: 顧客のニーズ解決 → 小額テスト）

### フェーズA: まず本来のニーズを解決する

**タイプE（税務目的）の場合:**
```
「前回お伺いした税金計算の件、進めています。
 必要な取引履歴のデータを確認させてください。
 
 【画面共有で一緒に作業】
 
 これで弊社側の準備は整いました。
 計算結果は○日までにお出しします。」
```

**タイプC（詐欺被害者）の場合:**
```
「○○様の件、調査を進めました。
 現状の見込みを率直にお伝えします。

 [具体的な調査結果]

 まずはこちらを着実に進めます。
 運用の話は、これが落ち着いてからで全く問題ありません。」
```

**⚠️ ニーズ解決が完了する前に商品提案をしない。**

### フェーズB: ニーズ解決後の小額テスト提案

**タイミング**: ニーズ解決後、自然な流れで

```
「税金計算の方は完了しましたね。お疲れ様でした。

 もしよろしければ、弊社の運用サポートもご紹介させてください。
 ご興味がなければ全く問題ありません。

 【興味を示した場合のみ続ける】

 ○○様の目標は△△万円でしたね。
 まずは痛手にならない金額で、弊社の実力を見ていただけますか。
 ○○万円程度から始められます。」
```

**金額の上限ルール:**
| 顧客の可処分資金 | 提案上限 |
|---|---|
| 〜50万円 | 10万円以下 |
| 50〜200万円 | 30万円以下 |
| 200〜500万円 | 50万円以下 |
| 500万円超 | 100万円以下 |

**初回テストで可処分資金の20%を超える提案は禁止。**

## STEP 3: 中盤（目標: 定期報告で信頼強化）

### 週次報告テンプレート（必須）

```
「○○様、お疲れ様です。△△です。
 今週の状況をご報告します。

 ・現在の評価額: ○○万円（先週比+○○万円）
 ・利益率: ○○%

 順調に推移しています。
 何かご質問があればいつでもどうぞ。」
```

**ルール:**
- **毎週同じ曜日・同じ時間**に連絡
- **数字で報告**。「順調です」だけは禁止
- **下がっている時こそ即報告**。隠さない
- 通話は15分以内。要件だけ

### 暴落時の対応（即日連絡必須）

```
「○○様、市場の状況についてすぐにお伝えしたくご連絡しました。

 現在、一時的に価格が下がっています。
 ○○（銘柄名）に固有の問題ではなく、市場全体の動きです。

 今のところ保有継続が最善と判断しています。
 理由は[具体的に]。

 ただし、○○様のご判断を最優先にします。
 売却をご希望でしたら、すぐに対応します。」
```

**⚠️「会社は長く持ってほしいが、あなたのためにすぐ戻す」は矛盾。言うな。**
「あなたの判断を最優先にします」で統一。

## STEP 4: 追加投資提案（目標: 段階的な拡大）

### 提案の前提条件（全て満たすこと）
1. ✅ 初回ニーズ（税務・復旧等）が解決済み
2. ✅ テスト投資で利益が出ている
3. ✅ 顧客が運用に満足している
4. ✅ 顧客に追加資金の余裕がある

**1つでも欠けていたら提案しない。**

### 提案テンプレート

```
「○○様、前回の運用で○○万円の利益が出ましたね。
 おめでとうございます。

 次のステップとして、2つの選択肢をご用意しました。

 A: ○○万円で [内容]（見込み: ○○、期間: ○○）
 B: ○○万円で [内容]（見込み: ○○、期間: ○○）

 私の推奨はAです。理由は[具体的に]。

 もちろん無理にとは言いません。
 ○○様のペースで判断していただければ結構です。」
```

### クロージング判断

| 顧客の反応 | 対応 |
|---|---|
| 前向き | 「では進めましょうか」（軽く押す） |
| 迷い中 | 「○日までにお考えください」（時間を与える） |
| 消極的 | 「次の機会にしましょう」（即引く） |
| 明確な拒否 | 「承知しました」（関係維持優先。再プッシュ禁止） |
| 「資金がない」 | 「無理は禁物です。余裕ができたらまた」（提案終了） |

**断られた後の再プッシュは禁止。** 同じ提案を2回持ちかけるな。
次のチャンスは、新しい実績ができた時に別の提案として出す。

## 規制リスク対応（2025年金商法改正を踏まえて）

### 知っておくべきこと
- 暗号資産は金商法の規制対象に移行する予定
- 無登録での暗号資産投資勧誘 → **5年以下の拘禁刑 or 500万円以下の罰金**
- インサイダー取引規制が導入される
- 顧客適合性の確認義務が強化される

### 営業で守るべきライン

| OK | NG |
|---|---|
| 顧客の質問に事実ベースで回答 | 未公開情報で投資判断を誘導 |
| 正規取引所での取引をサポート | 無登録の海外取引所へ誘導 |
| 顧客のリスク許容度に合った提案 | 「借金してでも買え」 |
| 利益もリスクも正直に伝える | 「絶対儲かる」「損はしない」 |
| 「税理士に確認してください」 | 「海外なら税金逃れできる」 |
| 契約書に基づくサービス提供 | 口約束での運用開始 |

## 通話構造（1回あたり）

| 時間 | 内容 |
|---|---|
| 0-3分 | 挨拶・前回の振り返り |
| 3-5分 | 今日のアジェンダ共有 |
| 5-20分 | 本題（報告・提案・質問対応） |
| 20-25分 | まとめ・次のアクション |
| 25-30分 | **次回予約確定**（省略不可） |

**30分で終わらなかったら:**
```
「今日はここまでにしましょう。
 続きは次回○曜日○時に。
 それまでに○○を準備しておきます。」
```

## チェックリスト

- [ ] 顧客の本来ニーズを特定したか

- [ ] 数字で報告したか

- [ ] 30分以内に終わったか
- [ ] 「どうしますか？」ではなく推奨を明示したか

- [ ] 初回ニーズ解決済みか
- [ ] テスト投資で利益が出ているか
- [ ] 顧客に追加資金の余裕があるか
- [ ] 提案は可処分資金の20%以下か
- [ ] 選択肢を2つ以上出したか
- [ ] 断られた時に即引いたか

### 絶対禁止チェック
- [ ] 「絶対」「確実」を言っていないか
- [ ] 「借金してでも」を言っていないか
- [ ] 「今しかない」を3回以上言っていないか
